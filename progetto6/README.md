Procedendo in modo iterativo e incrementale, applicando la metodologia TDD, sviluppate un modulo 
che implementa il sistema gemellare per la gestione dinamica della RAM presentato a lezione,
secondo le specifiche descritte nel file header:

ram.h

Successivamente apriremo un quiz Moodle dove potrete incollare la vostra soluzione, sottoporla a test automatici e inviarla ai docenti.
Nota: la ricerca di un nodo libero deve *obbligatoriamente* avvenire dando priorità al
figlio di sinistra (lbuddy). Questo di per sè non sarebbe necessario, ma semplifica i nostri test.
